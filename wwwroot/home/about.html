About the project

SmartSpend AI được thiết kế để làm gì?

Đây là web quản lý chi tiêu cá nhân hướng desktop-first, tập trung vào trải nghiệm nhập liệu nhanh, trực quan hóa dữ liệu tài chính và cảnh báo real-time đủ thuyết phục cho một đồ án năm 3 nhưng không quá phức tạp như sản phẩm doanh nghiệp.

Mục tiêu của nhóm

  • Làm rõ problem-solution: người dùng khó kiểm soát tiền khi ghi chép thủ công quá chậm.
  • Tạo điểm nhấn AI thực tế: rule-based parser và forecast đơn giản nhưng có giá trị demo.
  • Trình bày kiến trúc backend, auth, database, SignalR, phân quyền một cách có hệ thống.

Tech stack chính

ASP.NET Core Web API Entity Framework Core SQL Server JWT Authentication SignalR SMTP Gmail OTP Docker
A

Backend

Auth, OTP, role, CRUD Wallet/Transaction/Budget, admin API, audit log, alert hub.

B

Database

Schema mới cho Users, Roles, Wallets, Transactions, Budgets, BudgetAlerts, Keywords, AuditLogs.

C

Frontend

Landing page, auth pages, dashboard tài chính, admin pages và dark/light mode đồng bộ.

Database và phân quyền

SmartSpend tách rõ quyền Guest, Standard User, System Admin. Mỗi transaction, wallet và budget đều gắn với user hiện tại để đảm bảo data isolation. Bảng Keywords được dùng cho AI categorization theo rule-based dictionary, còn BudgetAlerts lưu các cảnh báo đã phát sinh để user xem lại trên dashboard.
